# داشبورد HOMEZY لمالكي العقارات 🏠

## نظرة عامة
داشبورد عصري وقابل للاستخدام لإدارة عقارات تأجيرية قصيرة الأجل. مبني بـ React و Tailwind CSS و Recharts.

## الميزات الرئيسية

### 📊 الصفحات المتوفرة

#### 1. **الصفحة الرئيسية** (`/`)
- عرض ملخص شامل لجميع العمليات
- بطاقات إحصائية تظهر:
  - إجمالي العقارات
  - الحجوزات النشطة
  - الأرباح الكلية
  - التقييم العام
- جدول الحجوزات الأخيرة
- رسم بياني للإيرادات الشهرية

#### 2. **إدارة العقارات** (`/properties`)
- عرض جميع العقارات بتصميم بطاقات حديثة
- إمكانية إضافة عقار جديد
- خيارات التعديل والحذف
- شريط بحث للتصفية السريعة
- عرض حالة كل عقار (متاح/مأجور)

#### 3. **الحجوزات** (`/bookings`)
- جدول شامل لجميع الحجوزات
- عرض بيانات الضيوف والتواريخ والأسعار
- تصفية الحجوزات حسب الحالة
- تحميل التقارير
- عرض حالة كل حجز (مؤكد/معلق/ملغى)

#### 4. **الإحصائيات والتحليلات** (`/analytics`)
- رسوم بيانية متقدمة:
  - الإيرادات مقابل عدد الحجوزات
  - معدل الإشغال (Pie Chart)
  - أداء العقارات (Bar Chart)
- معلومات إحصائية مهمة:
  - متوسط سعر الليلة
  - إجمالي الإيرادات
  - متوسط التقييم

#### 5. **الأرباح والعائدات** (`/profits`)
- عرض الأرباح الشهرية بالتفصيل
- جدول يوضح:
  - الإيرادات
  - المصروفات
  - صافي الأرباح
  - نسبة الربح
- بطاقات إحصائية للأرباح الكلية والمتوسطة

#### 6. **التعليقات والتقييمات** (`/comments`)
- عرض تعليقات الضيوف
- عرض التقييمات النجومية
- إحصائيات التقييمات (متوسط التقييم، نسبة الإيجابية)
- خيارات الرد والحذف والإعجاب

#### 7. **تعديل العقار** (`/properties/edit`)
- نموذج شامل لتعديل بيانات العقار:
  - المعلومات الأساسية (الاسم، الوصف، الموقع)
  - التسعير والتفاصيل
  - عدد الغرف والحمامات والمساحة
  - المرافق المتاحة
- إدارة الصور
- تحديث الحالة

## المكونات المستخدمة

### المكتبات الرئيسية
- **React** - مكتبة واجهات المستخدم
- **React Router** - التنقل بين الصفحات
- **Tailwind CSS** - التنسيق والتصميم
- **Lucide React** - الأيقونات الحديثة
- **Recharts** - رسم البيانات والرسوم البيانية

### هيكل المشروع
```
src/
├── components/
│   ├── layout/
│   │   ├── Navbar.jsx      - شريط التنقل العلوي
│   │   ├── Sidebar.jsx     - القائمة الجانبية
│   │   └── Layout.jsx      - المخطط العام
│   └── shared/
│       ├── StatCard.jsx    - بطاقة الإحصائيات
│       ├── RecentBookingsTable.jsx  - جدول الحجوزات
│       └── RevenueChart.jsx - رسم الإيرادات
├── pages/
│   ├── HomePage.jsx
│   ├── PropertiesPage.jsx
│   ├── BookingsPage.jsx
│   ├── AnalyticsPage.jsx
│   ├── ProfitsPage.jsx
│   ├── CommentsPage.jsx
│   └── PropertyEditPage.jsx
├── App.js               - التوجيه الرئيسي
└── index.css            - الأنماط العامة
```

## كيفية التشغيل

### المتطلبات
- Node.js (النسخة 16 أو أحدث)
- npm أو yarn

### التثبيت
```bash
cd dash-onwer
npm install
```

### تشغيل التطوير
```bash
npm run dev
```

الموقع سيكون متاحاً على `http://localhost:5174/`

### الإنتاج
```bash
npm run build
npm run preview
```

## الميزات التصميمية

### نظام الألوان
- **الأزرق الرئيسي**: #3b82f6 - للعناصر الفعالة والزر الرئيسي
- **الأخضر**: #10b981 - للحالات الإيجابية (متاح، مؤكد)
- **الأحمر**: #ef4444 - للتنبيهات والحذف
- **البرتقالي/الأصفر**: للحالات المعلقة
- **الرمادي**: للنصوص والعناصر الثانوية

### سهولة الاستخدام
- **دعم النصوص العربية** كاملاً مع اتجاه RTL
- **تصميم سريع الاستجابة** يعمل على جميع الأجهزة
- **واجهة بديهية** سهلة الاستخدام
- **رموز واضحة** من مكتبة Lucide

### التفاعلات
- **الرسوم المتحركة الناعمة** عند التمرير والنقر
- **الظلال** لإضافة العمق البصري
- **الانتقالات الناعمة** بين الحالات

## نصائح للتطوير

### إضافة صفحة جديدة
1. أنشئ ملف `.jsx` جديد في مجلد `pages/`
2. استخدم مكون `Layout` لتطبيق التخطيط الموحد
3. أضف المسار في `App.js`

### إضافة مكون مشترك
1. أنشئ الملف في `components/shared/`
2. قم بتصديره من الملف الرئيسي
3. استخدمه في الصفحات المختلفة

### تعديل الألوان والأنماط
جميع الأنماط تستخدم Tailwind CSS، يمكن تعديل الألوان مباشرة في className
